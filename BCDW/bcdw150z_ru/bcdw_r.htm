 Что это?
Это менеджер загрузки с компакт-дисков.
  • Позволяет создавать мульти-загрузочные (Multiple-Image Bootable CD) компакт-диски для компьютеров, которые не поддерживают мульти-загрузку с компакт-дисков.
  • Позволяет создавать мульти-язычное, иерархическое меню загрузки с настраиваемым внешним видом.
  • Поддерживает расширенный набор форматов загрузочных образов (примеры).

 Как записать мульти-загрузочный компакт-диск с BCDW?
  • Создайте временную папку для файлов Вашего мульти-загрузочного компакт-диска (например, C:\MyBootCD\)
  • Распакуйте ZIP-архив с BCDW в эту папку
  • Добавьте в эту папку Ваши файлы (загрузочные образы и пр.)
  • Откройте с помощью программы Windows Notepad (Блокнот Windows) файл C:\MyBootCD\BootCat.Ini
  • Измените, каталог вариантов загрузки по Вашему усмотрению (удалите лишние варианты/добавьте свои)
  • Запустите Вашу программу для записи компакт-дисков
  • Создайте проект загрузочного компакт-диска со следующими параметрами:
    Загрузочный образ: C:\MyBootCD\bcdwboot.bin
    Режим загрузки: No emulation (Без эмуляции)
    Адрес загрузки: 07c0
    Количество секторов для загрузки: 4
    (как это сделать для Nero, CDRWin, Easy CD Creator, Instant CD+DVD, WinOnCD, CDImage или mkisofs)
  • Добавьте в проект содержимое папки C:\MyBootCD\
  • Запишите диск
 Настройка BCDW
  • Настройка BCDW заключается с создании двух конфигурационных файлов (или изменении готовых):
    • \BCDW\bcdw.ini с параметрами запуска программы
    • \BootCat.Ini с каталогом вариантов загрузки и описанием внешнего вида интерфейса
  • Оба файла являются текстовыми файлами и могут быть созданы с помощью программы Windows Notepad (Блокнот Windows), или аналогичной.
  • Конфигурационные файлы могут быть не только текстовыми файлами DOS/Windows, но и текстовыми файлами *NIX/Linux.
  • Конфигурационные файлы имеют структуру схожую со структурой *.ini/*.inf-файлов Windows:
    • файлы разбиты на секции, каждая из которых начинается с имени секции в квадратных скобках
    • секции содержат строки вида "ключевое слово = значение", с помощью которых задаются различные настройки программы
    • все пустые строки и строки, начинающиеся с символа ';' (точка с запятой) игнорируются
  • Замечания:
    1. Текущая версия BCDW использует только секцию [Options] остальные секции (если есть) игнорируются (кроме этого, BCDW игнорирует неизвестные ключевые слова в секции [Options])
    2. В конфигурационном файле \BootCat.Ini так же используется пространство от начала файла до первой секции под каталог вариантов загрузки (см. описание файла \BootCat.Ini)
 Конфигурационный файл \BCDW\bcdw.ini
Конфигурационный файл \BCDW\bcdw.ini может содержать секцию [Options] с четырьмя ключевыми словами (а может и не содержать - все четыре ключевых слова и сама секция [Options] являются необязательными, т.е. простейший файл \BCDW\bcdw.ini это пустой файл).
 Пример файла \BCDW\bcdw.ini.
[Options]
Password = secret
Logo     = \BCDW\logo.gif
Font     = \BCDW\FONTS\russian.fnt
CharSet  = \BCDW\CHARSETS\cp1251r.cpf
RootMenu = \BootCat.Ini
 Ключевые слова в файле \BCDW\bcdw.ini
Password (или PasswordMD5Hash)  Задает пароль (или его MD5 Hash) на запуск BCDW.
Logo  BCDW, при запуске, может вывести на экран графическую заставку.
Ключевое слово Logo указывает BCDW имя и расположение *.gif файла с этой заставкой.
Файл с заставкой может быть анимационным *.gif файлом.
Файл с заставкой не может иметь размер более ~500Kb.
BCDW может выводить заставки в одном из следующих видеорежимов: 320x200x256, 640x480x256, 800x600x256, 1024x768x256, 1280x1024x256 и 1600x1200x256 (использует ближайший).
Если картинка не помещается на экране, то она центрируется и усекается по краям.
Если ключевое слово Logo отсутствует, то заставка не выводится.

Font  При включении компьютера (или после перезагрузки), в видеоадаптер загружен стандартный английский (cp437) шрифт 8x16.
BCDW, при запуске, позволяет загрузить в видеоадаптер шрифт с другим набором символов (например, с символами русского алфавита) и/или шрифт с другой высотой символа (от высоты символа зависит количество строк на экране).
Ключевое слово Font указывает BCDW имя и расположение файла с шрифтом, который необходимо загрузить.
Размер файла с шрифтом может быть в пределах от 1024 байт (шрифт 8x4 - 100 строк на экране) до 8192 байт (шрифт 8x32 - 12 строк на экране).
Если ключевое слово Font отсутствует, то BCDW использует стандартный шрифт видеоадаптера.

CharSet  В конфигурационном файле \BootCat.Ini вы можете использовать тексты и имена файлов не только на английском языке, но и на некоторых других национальных языках (например, русском).
Для набора текстов на национальных языках, в различных операционных системах используются различные кодировки
(например, для русского языка, как правило, используются следующие кодировки: в DOS - cp866, в Windows - cp1251, а в *NIX/Linux - KOI8-R).
Для корректной обработки этих текстов и имен файлов (вывода на экран, поиска файлов на компакт-диске и т.п.) BCDW необходимо знать, какую кодировку Вы использовали при наборе файла \BootCat.Ini.
Ключевое слово CharSet указывает BCDW имя и расположение файла с описанием использованной кодировки.
Если ключевое слово CharSet отсутствует, то BCDW считает, что в файле \BootCat.Ini Вы использовали только английские тексты и имена файлов.

RootMenu  По умолчанию, BCDW ищет описание загрузочного меню в файле \BootCat.Ini.
Вы можете изменить имя и расположение этого файла (например, на \BCDW\BootCat.Ini).
Ключевое слово RootMenu указывает BCDW, какой файл необходимо использовать.
 Конфигурационный файл \BootCat.Ini
Файл \BootCat.Ini (и другие файлы с описаниями меню) состоит из двух частей:
  • Каталога вариантов загрузки - строк между началом файла и первой секцией в файле (или концом файла, если секций в файле нет)
  • Необязательной секции [Options] с описанием внешнего вида интерфейса

 Каталог вариантов загрузки в файле \BootCat.Ini
Каталог вариантов загрузки представляет собой набор строк, каждая из которых описывает один вариант загрузки. Каждая строка состоит из нескольких частей (в текущей версии BCDW - из четырех) разделенных символом ';' (точка с запятой):

часть 1 ; часть 2 ; часть 3 ; часть 4

  1. Имя файла с описанием подменю или имя файла с загрузочным образом или управляющая команда BCDW
  2. Название варианта загрузки
  3. Текст комментария к варианту загрузки ( ^ - разделитель строк) или ключевое слово из секции [Options] с текстом комментария
  4. Параметр для загрузочного образа или управляющей команды BCDW
Все части описания, кроме первой, являются необязательными.
 Имена и формат файлов с описаниями подменю
Файлы с описаниями подменю могут иметь любые *.ini имена. Формат этих файлов идентичен формату файла \BootCat.Ini.
 Имена и форматы файлов с загрузочными образами
Файлы с загрузочными образами могут иметь любые имена, кроме *.ini.
BCDW поддерживает следующие форматы загрузочных образов:
  • Образы дискет и логических дисков произвольного объема (до 8Gb) с файловой системой FAT(12/16/32). Например, *.ima файлы создаваемые программой WinImage. (примеры)
  • Образы дискет емкостью 160Kb, 180Kb, 320Kb, 360Kb, 720Kb, 820Kb, 1.2Mb, 1.44Mb, 1.68Mb, 1.72Mb, 2.88Mb с файловой системой отличной от FAT
  • Образы жестких дисков с произвольным количеством разделов (примеры)
  • Программы с адресом загрузки 07c0:0000 (например, загрузочные сектора с компакт-дисков Microsoft Windows XP/2000/NT nt5boot.bin, nt4boot.bin, w2ksect.bin, xpboot.bin и т.п.)
  • Программы с адресом загрузки 07c0:0000 и таблицей "Boot Info Table" (например, программа isolinux.bin)
  • Загрузчик программы установки Microsoft Windows XP/2000/NT (файл \i386\setupldr.bin)
 Управляющие команды BCDW
Вместо имени файла с загрузочным образом, Вы можете указать управляющую команду BCDW. Текущая версия BCDW поддерживает следующие команды:

A:\ Загрузиться с реального дисковода. Команда не требует параметра.
C:\ Загрузиться с реального жесткого диска. Команда не требует параметра.
:Reboot Перезагрузить компьютер. Команда не требует параметра.
:Restart Перезапустить BCDW с другими настройками. Команда требует параметр - имя файла с настройками (имя другого \BCDW\bcdw.ini).
:Return Вернуться из подменю. Команда не требует параметра.
:Nop Ничего не делать. Команда не требует параметра.
: То же самое, что и :nop, но вариант загрузки с этой командой невозможно выбрать (недоступный пункт меню). Команда не требует параметра.
 Параметры для загрузочных образов
Загрузочные образы не поддерживают прием параметров, но с помощью BCDW Вы можете передать параметры некоторым из них, а именно:
  1. Специальному загрузочному образу dos4bcdw.ima (как создать этот образ?) - имя DOS-программы на компакт-диске для запуска
  2. Программе ISOLinux - команду для выполнения (подробности)
  3. Программе CD Shell - команду для выполнения (подробности)
  4. Программе BootScriptor - команду для выполнения (подробности)
  5. Программе diskem1x.bin - команду для выполнения (подробности)

 Как переписать программу с загрузочного компакт-диска на компакт-диск с BCDW?
Некоторые программы позволяют создать загрузочные компакт-диски (например, программы Acronis).
Для того, что бы переписать такие программы на компакт-диск с BCDW сделайте следующее:
  1. Перепишите все файлы и папки с загрузочного компакт-диска на компакт-диск с BCDW (на компакт-дисках Acronis файлов нет - ничего переписывать не надо)
  2. С помощью WinISO создайте *.iso образ загрузочного компакт-диска (пункт "Make ISO from CDROM" в меню "Actions")
  3. Откройте созданный *.iso образ (пункт "Open" в меню "File")
  4. Сохраните загрузочную информацию в *.wbt файл (пункт "Save boot information to file..." в меню "Bootable CD")
  5. Поместите созданный *.wbt файл на компакт-диск с BCDW
  6. Добавьте в \BootCat.ini описание варианта загрузки аналогичное этому:

    \acronis.wbt ; Программа Acronis

Описание с иллюстрациями см. здесь

Да можно. Для этого сделайте следующее:
  1. Поместите файл diskem1x.bin в произвольную папку на компакт-диске (например, в /)
  2. Убедитесь, что на компакт-диске нет файла (если он есть - удалите его):

    /diskemu.cmd

  3. Используйте в /BootCat.Ini описания вариантов загрузки подобные этим:

    /diskem1x.bin ; Командная строка diskem1x.bin
    /diskem1x.bin ; Sample script ;; batch diskem1x.cmd
 "Press any key to boot from CD..."
Если Вы хотите, что бы перед запуском BCDW появлялось приглашение "Press any key to boot from CD...", то перепишите файл bootfix.bin из дистрибутива Microsoft Windows XP/2000 в папку с BCDW.
 Быстрая отмена загрузки с компакт-диска с BCDW
BCDW при запуске проверяет, нажата ли левая клавиша Alt и если она нажата, сразу переходит к загрузке с жесткого диска без вывода каких либо сообщений, заставок и т.п.
